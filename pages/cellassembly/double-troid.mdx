## ダウンロード

実行ファイル、パラメータファイル (par.txt) および焼結ダイヤモンドアンビルの attenuation coefficient data（`sc_att_smooth.dat`：測定データを 10 Å まで外挿、さらにスムージングしたもの）

- `Atten4powDT.zip` (2014/01/19 ver.)
- ソースコード（実行ファイルも含む）: `Atten4powDTsource19Jan2014.zip` (2014/01/19 ver.)
- 光路長計算の解説PDF: `Double troidal anvil の光路長計算.pdf`

---

## 使い方

1. 以下の3つのファイルを適当なフォルダに入れる。

   `Atten4powDT.exe`, `par.txt`, `sc_att_smooth.dat`

2. `par.txt` を必要に応じて編集

### par.txt の中身

```
10               : サンプリングポイントの1軸あたりの数（10で十分。5程度でもよい）
3                : 形状パラメータの数（double toroidal anvilの場合は3）
7.75             : gp(1), 外側のgasket (ring)の半径 [mm]*
0.4              : gp(2), gasket最薄部の厚さ [mm]*
0.8              : gp(3), anvil gap [mm]*
85 95 1          : gammastart、gamma end、gamma step [deg]
0 0 1            : nustart, nu end, nu step [deg]
sc_att_expo.dat  : アンビルの吸収係数情報のあるファイル名（拡張子つき）
0 0              : 試料の減弱係数 [mm⁻¹]（μa, μb：μsample = μa * λ + μb）
400 100          : bin数、tofのステップ幅 [μsec]
26.5             : L（PLANET → 26.5, TAKUMI → 42, NOVA → ?）
test0.8.dat      : 出力ファイル名（拡張子つき）
```

（`*` gp(1)〜(3)がどこの部分を表すかわからない場合は、「Double toroidal anvil の光路長計算.pdf」を見てください。）

3. ファイルのあるフォルダでコマンドプロンプトを開く（Windows 7 の場合、アドレスバーで `cmd` とタイプ）

4. コマンドプロンプト上で

   ```
   Atten4powDT.exe < par.txt
   ```

   とタイプすると、実行する。

出力ファイルは、`tof, A(=I/Io)` となっている。
