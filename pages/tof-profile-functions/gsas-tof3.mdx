## プロファイル関数の構成

最終的なプロファイル関数 $H(\Delta T)$ は、４つのピーク関数の足し算になっています。

- **HG1:** $G(\Delta T)$ と $a\exp(a\Delta T)$ のコンボリューション
- **HG2:** $G(\Delta T)$ と $b\exp(-b\Delta T)$ のコンボリューション
- **HL1:** $L(\Delta T)$ と $a\exp(a\Delta T)$ のコンボリューション
- **HL2:** $L(\Delta T)$ と $b\exp(-b\Delta T)$ のコンボリューション

それぞれよく似ているので、パラメータ間の相関はかなり強いです。

---

## ピーク中心とΔTの定義

$T_{ph}$: ある相 $p$、あるヒストグラム $h$ における時間 $T$ という意味

$$T_{ph} = \text{DIFC}\, d_p + \text{DIFA}\, d_p^2 + \text{ZERO}$$

$$\Delta T = t - T_{ph}$$

## 特殊関数の補足

ここで使われている特殊関数について以下に補足します。

**erfc:** 相補誤差関数 (complementary error function)

$$\mathrm{erfc}(x) = 1 - \mathrm{erf}(x)$$

**$E_1$:** (1次の) 指数積分 (exponential integral)

また、$\mathrm{Im}[e^p E_1(p)]$ は $e^p E_1(p)$ が複素数なので、その虚数部、という意味です。

複素指数関数 $e^z$ [$z = x + iy$, $x$ と $y$ は実数] は、

$$e^z = e^x(\cos y + i\sin y)$$

と展開して計算できます。

$\mathrm{erfc}(x)$ [$x$: 実数] は、Excel のワークシート関数にもあり、比較的簡単に使えますが、$E_1(z)$ [$z$: 複素数] はなかなか実装しているプログラムがありません。しかし、さすがに歴史のある FORTRAN では、いくつか実装例があります。

## 数値計算上の注意

$\alpha$ の値が大きいと、$u$ の値が大きくなりすぎ、$\exp(u)$ は非常に大きな値になるか、オーバーフローします。一方、$\mathrm{erfc}(y)$ はほとんど 0 に近い値（あるいはアンダーフロー）になり丸め誤差の原因になります。

そこで、スケーリング相補誤差関数 $\mathrm{erfcx}(x)$

$$\mathrm{erfcx}(x) \equiv e^{x^2}\mathrm{erfc}(x)$$

を使って、

$$\exp(u)\,\mathrm{erfc}(y) = \exp(u - y^2)\,\mathrm{erfcx}(y)$$

と置き換えると、この問題を回避できます。$\mathrm{erfcx}(x)$ は多くのプログラム言語でサポートされています。
